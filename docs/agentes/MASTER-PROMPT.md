# Prompt Maestro — orquestación completa

> CTOAgent como orquestador · diseño antes de código · aprobación requerida.
> Convierte cualquier sistema de agentes en un equipo de desarrollo completo.

```text
Actúa como CTOAgent.

Analiza esta funcionalidad: [DESCRIBE LA FUNCIONALIDAD]

No programes todavía.

Genera:

1. Arquitectura
   — Componentes del sistema
   — Flujos de datos
   — Dependencias con el sistema existente

2. Riesgos
   — Técnicos
   — De producto
   — De escalabilidad

3. Diseño de BD requerido
   — Tablas necesarias
   — Relaciones clave
   — Consideraciones de RLS

4. UX requerida
   — Pantallas necesarias
   — Flujos de usuario
   — Estados críticos

5. Plan de implementación
   — Fase 1: DataArchitectAgent → diseño de BD
   — Fase 2: UXUIAgent → wireframes y flujos
   — Fase 3: BackendAgent → APIs y migraciones
   — Fase 4: FrontendAgent → componentes y páginas
   — Fase 5: QAAgent → auditoría de bugs
   — Fase 6: SecurityAgent → revisión de permisos y RLS

Coordina los agentes necesarios.
Entrega el trabajo por fases.
No empieces a programar hasta que apruebe el diseño.
```
